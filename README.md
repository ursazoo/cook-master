# cook-master

## 项目描述

根据所选的食材对菜品进行筛选，并提供菜品的做法。

同时提供了管理后台，用于对食材、菜谱进行管理。

## 前端技术栈

| 前端框架     | Vue 3.2 + TypeScript 4.9                                                          |
| ------------ | --------------------------------------------------------------------------------- |
| UI 框架      | [Naive UI](https://www.naiveui.com/)                                              |
| CSS 扩展     | [SCSS](https://sass-lang.com/)                                                    |
| 状态管理工具 | [Pinia](https://pinia.vuejs.org/)                                                 |
| 构建工具     | [Vite](https://cn.vitejs.dev/)                                                    |
| 包管理工具   | [pnpm](https://www.pnpm.cn/)                                                      |
| 格式规范     | [eslint](https://eslint.org/) + [f2elint](https://www.npmjs.com/package/f2elint/) |
| Git 工作流   | [husky](https://typicode.github.io/husky/#/)                                      |
