# menu-master

## 项目描述

根据所选的食材对菜品进行筛选，并提供菜品的做法。

同时提供了管理后台，用于对食材、菜谱进行管理。

## 前端技术栈

| 前端框架     | React18.2 + TypeScript4.9 |
| ------------ | ------------------------------------------------------------------------------- |
| UI框架       | [Ant Design](https://ant.design/index-cn)                                       |
| css扩展      | [Less](https://lesscss.org/)                                                    |
| 状态管理工具 | [Redux](https://cn.redux.js.org/)                                                 |
| 构建工具     | [Vite](https://cn.vitejs.dev/)                                                   |
| 包管理工具   | [pnpm](https://www.pnpm.cn/)                                                      |
| 格式规范     | [eslint](https://eslint.org/) + [f2elint](https://www.npmjs.com/package/f2elint/) |
| Git工作流    | [husky](https://typicode.github.io/husky/#/)                                      |
